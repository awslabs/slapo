SampleModule example¶

This example will demonstrate the power function and class_power from our package ‘SampleModule’.

import SampleModule.module

SampleModule.module.fun_power(2,3)

8

The function power returns the first number raised to the power of the second number.

my_class = SampleModule.module.class_power(2,3)
my_class.power()

8

Instances of power_class have a power method. If a non-number is used when initiating the class, an informative statement is printed:

my_class = SampleModule.module.class_power('a',3)
my_class.power()

Something went wrong. Make sure x and y are both numbers
