class slapo.op.dropout.DropoutWithTensorParallel(p=0.5, inplace=False)[source]¶

The dropout that supposed to be used in parallel region. In parallel region means the original input tensor is partitioned due to tensor parallelism or sequence parallelism. In this case, we need to make sure the dropout on each device in the same tensor parallel group has DIFFERENT random seed; otherwise each partitioned tensor will have the same dropout mask, which may hurt the convergence.
